Ice Cream Inventory & Reporting System

A production-level inventory and reporting system built for a multi-branch ice cream company operating across 10 branches.

The system centralizes weekly inventory reporting and replaces manual Excel consolidation with automated multi-branch reporting.

🚀 Features

Secure employee authentication (Firebase Authentication)

Role-based access control (10 branch users + 1 manager)

Branch-level permission enforcement (Firestore Rules)

Weekly inventory submission forms

Automated multi-sheet Excel export (one sheet per branch)

Automatic Merrycream-to-KG conversion logic

Free quantity exclusion from totals

Dynamic pricing calculations

Clean structured reporting layout

🛠 Tech Stack

React.js

Firebase Authentication

Firestore

ExcelJS

Vite

📊 System Architecture

Role-based system (Employee / Manager)

Firestore security rules for data integrity

Modular component-based frontend architecture

📌 Status

Currently used in real operations and employee onboarding.
